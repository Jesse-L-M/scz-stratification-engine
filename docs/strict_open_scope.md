# strict-open v0 scope

## In scope now

- A public cognitive stability and trial-noise audit engine for schizophrenia cohort-integrity work.
- Claim-boundary, source, and execution-stack documentation for `strict-open v0`.
- A default public core path centered on `ds005237 / TCP`.
- Success criteria framed as technical feasibility, credibility, cohort-integrity signal, and de-risking for later work.

## Out of scope now

- Sponsor-grade enrichment or commercial positioning.
- Treatment recommendation or treatment-response claims.
- Persistent cognitive burden claims under real medication follow-up.
- HCP-YA, gated datasets, or any gated-data-dependent result.
- Novel biological subtype discovery claims.
- Recasting this public build as the final product.

## Relationship to the later gated-data product

`strict-open v0` is the public predecessor to a later gated-data product, not a substitute for it. The public audit build is meant to clarify the technical path and de-risk future work, while the later gated-data product can pursue claims and evidence that are explicitly outside the scope of `strict-open v0`.
