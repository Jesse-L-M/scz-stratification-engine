# tcp-ds005237

Primary sources used for this audit:

- https://openneuro.org/datasets/ds005237
- https://raw.githubusercontent.com/OpenNeuroDatasets/ds005237/1.1.3/README
- https://raw.githubusercontent.com/OpenNeuroDatasets/ds005237/1.1.3/participants.tsv

Current call:

- `poor_functional_outcome` is potentially benchmarkable via LIFE-RIFT and MCAS.

Current limits:

- public participant metadata is group-level (`Patient` vs `GenPop`) rather than
  diagnosis-granular
- the public release does not confirm repeated follow-up visits
- one-year nonremission, persistent negative symptoms, and relapse proxies are
  not currently supported from public metadata
